Overlay Analysis

The objective of the project was to find suitable locations for building bush lodges fro an ecotourism organization in Krugrer National Park. Taken into conisideration were spatial multiple criteria.Several geoprocessing and spaitial analysist tools were used including Intersect, Zonal Statistics, Union and more. The result is a map showing suitable sites based on land cover, land use and proximity to rivers.

Multi-Thematic Map

This multi-thematic map created using ArcGIS dispays 2000 US Census data of the African American population using a choropleth, dot density and graduated symbol theme. This tecnique allows the reader to view data in a vareity of ways and scales on one map.

Hot Spot Analysis

The map above displays clustering "hotspots" of 311 calls requesting graffiti removal in Chicago to help personnel allocate resources. Using the Hot Spot Anaylsis tool in ArcGIS, I was able to calculate the spatial autocorrelation of 311 calls and interpret the statistical results given the overall z-score. The areas in red have a high clusting of 311 calls with 95% confidence rate.

Distance Analysis

The above map displays two distance analysis from the campus of Northeastern Illiniois University made from tools in ArcGIS. The left map shows a raster ring buffer of the euclidean distance from campus. On the right shows a vector based network distance from campus.
